
Welcome to my new blog! This space is dedicated to my thoughts, experiments, and journeys in software engineering, frontend architectures, and clean code practices.

### Why a Rewrite?

I previously started this portfolio site but quickly realized the underlying codebase was becoming cluttered and didn't follow the clean standards I strive for. So, I did what any software engineer loves to do: **I started over**.

This new project is built from the ground up using **Angular v22**, utilizing:

- **Signals** for reactive local state management.
- **Server-Side Rendering (SSR)** for fast first-meaningful-paint times.
- **Static Prerendering** to serve highly optimized static HTML directly from the edge.
- **ngx-translate** for legally required multi-language compliance.
